# Setup & Usage

## Getting Started

1. **Clone or download the repository.**
2. **Open `index.html` in your browser.**
3. **Navigate using the menu to explore all pages.**

## Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- No server or backend required
